DISPLAY FORMAT:

For each cart item, show:
1. Item name and quantity
2. Indented list of selected options below
3. Each option shows: option type name, value name, and price (if not zero)
4. Total item price includes all option prices

IMPLEMENTATION:

1. SAVE OPTIONS WITH CART ITEMS:
   When customer adds item to cart, save:
   - Base item info
   - Array of selected options with: option_type_name, option_value_name, option_price

2. DISPLAY IN CART:
   For each cart item, show:
   - Item name and quantity
   - Loop through selected options and display each one
   - Show option prices if greater than 0
   - Calculate total including all option prices

3. DISPLAY IN CHECKOUT:
   Same format as cart
   - Group by merchant if multi-merchant order
   - Show all items with their selected options
   - Clear itemized view of what customer is ordering

4. HANDLE DIFFERENT OPTION TYPES:
   - Single selection (Size, Flavor): Show selected value
   - Multiple selection (Add-ons): Show all selected values
   - Free options: Show name without price or show (₱0)

5. FORMATTING:
   Use clear visual hierarchy:
   - Item name: Bold, larger
   - Options: Indented, smaller font, gray color
   - Prices: Right-aligned

This allows customers to review exactly what they're ordering with all customizations visible before placing the order.
