Fix and improve the option type management when merchant adds menu items:

REQUIREMENTS:

1. WHEN MERCHANT ADDS OPTION TYPE:
   Automatically show input fields for:
   - Option type value (name)
   - Price for that value
   - Delete button for each value

2. AUTOMATIC FIELDS:
   When merchant selects or creates option type (e.g., "Size"), immediately show:
   - First value input field
   - Price input field
   - [+ Add Another Value] button
   - Delete icon for that value

3. ADDING MORE VALUES:
   When merchant clicks [+ Add Another Value]:
   - Add new row with value input and price input
   - Each row has its own delete button
   - Can add unlimited values

4. DELETING VALUES:
   - Each value row has delete icon/button
   - Click to remove that specific value
   - When LAST value is deleted â†’ Remove entire option type

5. DELETING OPTION TYPE:
   Two ways to remove:
   - Delete all values one by one (removes option type when last value deleted)
   - Or have separate "Remove Option Type" button that deletes entire section

6. REORDERING OPTION TYPES:
   - Add drag handles to each option type section
   - Merchant can drag to reorder option types
   - Order is saved and shown to customers
