1. LAYOUT:
   - 3-4 cards in horizontal row
   - Equal width
   - Slight gap between cards
   - Light colored backgrounds (yellow/green/blue tints)

2. CARD STRUCTURE:
   Each card contains:
   - Top: Label text (small, medium weight)
   - Middle: Large amount (bold, prominent)
   - Bottom: Description text (smallest, light gray)

3. STYLING:
   - Background: Light pastel colors
     * Yellow tint for merchant
     * Green tint for app
     * Blue tint for rider
   - Border: Rounded corners
   - Padding: Comfortable spacing
   - Typography hierarchy clear

   - Merchant cards: Light yellow/gold tint
   - App cards: Light green/teal tint
   - Rider cards: Light blue tint

6. TEXT FORMATTING:
   - Label: 14px, medium weight, dark gray
   - Amount: 24-28px, bold, primary color
   - Description: 12px, light weight, gray

7. RESPONSIVE:
   Desktop: All cards in one row
   Tablet: 2 cards per row
   Mobile: 1-2 cards per row (stack vertically)
